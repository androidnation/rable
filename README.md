## Usage

* _prototype functions_
  
    __Note: prototype function work as shown here: ```object.yourPrototypeFunction()```__
    
    - __.switch()__
      _applied on booleans only_
      ```javascript
        true.switch()     //true is switched to false
        false.switch()    //false is switched to true
      ```
